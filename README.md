# 🎬 Netflix Clone UI

A simple **Netflix-inspired UI  Clone** built using **HTML & CSS**.
This project replicates the look and feel of Netflix with movie sections, banners, and scrolling lists.

---

## 🚀 Features

* 🎥 Hero banner with featured movie
* 📺 Horizontal scrolling movie lists
* 🎭 Categories like:

  * Top Picks
  * Funny Movies
  * Horror Movies
  * Trending Movies
* ✨ Hover animations on buttons and images

---

## 🛠️ Technologies Used

* HTML5
* CSS3 (Flexbox, Hover Effects)
